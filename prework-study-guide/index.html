HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element is usally the behind the scenes information while the body is visible to the user.
  • h1 through h6 are used for different headings. h1 should only be used once and at the top of the page.
  • ul stands for unlisted while li stands for whats in the list.

CSS

  • A margin indictes how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Assigning colors in CSS can be done by semantic references or by the hexadecimal code.

Git

  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Variables sometimes hold values that have different data types such as strings, numbers or booleans
  • Operators are mathematical symbols the produce a result based on two values.
  • Functions are a way of packaging functionality that you wish to reuse. This is a good alternative to repeatedly writing the same code.
  • Events are the code structure that listen for activity in the browser. The most obvious event would be a click event when you click on something with your mouse.
